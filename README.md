# 🔵 Custom Arduino-Compatible Board (ATmega328P)

## 📌 Project Overview
This project is a custom-designed Arduino-compatible development board built using KiCad.  
The board is based on the ATmega328P microcontroller and supports Arduino IDE programming.

Designed as part of my PCB Design Portfolio.

---

## 🛠 Tools Used
- KiCad (Schematic & PCB Design)
- Arduino IDE
- Gerber Generation for Manufacturing

---

## 🔧 Hardware Specifications

- Microcontroller: ATmega328P-PU
- Operating Voltage: 5V
- Clock: 16 MHz Crystal
- USB-to-Serial Programming Support
- ICSP Header
- DC Barrel Jack Input
- Onboard 5V Voltage Regulator (7805)
- Power LED
- Reset Button
- Digital & Analog I/O Headers
